## Visual1 Starter Kit

1. cd {{ project_name }}
2. composer install
3. Create mysql database
4. Create `.env` file from `.env.example` and fill in database name, generate an app key, etc.
5. php artisan migrate --seed
6. npm install
7. npm run dev
